# 문자열 반복 (Python)

## 개념

- 문자열 × 정수 → 문자열을 해당 횟수만큼 반복하여 이어붙임

## 예시

```python
"*" * 1   # "*"
"*" * 3   # "***"
"ab" * 2  # "abab"
```

## 활용: BOJ 2438 (별 찍기 - 1)

```python
n = int(input())
for i in range(n):
    print("*" * (i + 1))
```

## 참고

- 공백 문자열도 동일하게 동작: " " \* 4 → 네 칸 공백
- 음수 곱은 빈 문자열 반환: "abc" \* -1 → ""
